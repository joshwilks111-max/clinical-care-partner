Clinical Care-Partner · Architecture

v3.1 · one chat route, a typed tool loop · claude-opus-4-7 · Vercel AI SDK 6 · Next.js 16

Judgment up, execution down. A thin clinical router that hands off to deterministic, safety-audited skills. The model reasons about the differential and picks a dose rule by id. The tools own every number. The seam between the skill and the tools is the whole design.
